# SEED Labs – Cybersecurity Lab Documentation

A collection of lab reports completed as part of the [SEED Project](https://seedsecuritylabs.org/) curriculum developed at Syracuse University, covering hands-on offensive and defensive cybersecurity concepts.
---
## Disclaimer

These reports are shared for educational and portfolio purposes only. The techniques documented here were performed in isolated lab environments. Do not use any of these methods on systems you do not own or have explicit permission to test.

---

## Labs

| # | Lab | Topics |
|---|-----|--------|
| 1 | [Spectre Attack](./Spectre%20Attack.pdf) | Side-channel attacks, CPU speculative execution, cache-timing |
| 2 | [Secret-Key Encryption](Secret-Key%20Encryption.pdf) | AES-CBC, AES-CFB, Blowfish, ECB weaknesses, OpenSSL |
| 3 | [Public Key Encryption](Public%20Key%20Encryption.pdf) | X.509 certificates, PKI, SSL/TLS, certificate authorities |
| 4 | [RSA](RSA.pdf) | RSA encryption/decryption, digital signatures, BigNum arithmetic |
| 5 | [Buffer Overflow](Buffer%20Overflow.pdf) | Stack exploitation, shellcode injection, NOP sleds, GDB |
| 6 | [Packet Sniffing and Spoofing](Packet%20Sniffing%20and%20Spoofing.pdf) | Scapy, ICMP/TCP spoofing, promiscuous mode, Wireshark |
| 7 | [VPN Tunneling](VPN%20Tunneling.pdf) | TUN interfaces, IP routing, tcpdump, encrypted tunnels |
| 8 | [Firewall Evasion](Firewall%20Evasion%20Lab.pdf) | SSH port forwarding, iptables, traffic analysis |

## Tools & Technologies

- **Languages:** Python, C
- **Cryptography:** OpenSSL, RSA, AES, Blowfish
- **Network Tools:** Scapy, Wireshark, tcpdump, iptables
- **Debugging:** GDB
- **Environment:** Linux (Ubuntu), Docker

---

## About SEED Labs

The SEED Project is a widely used cybersecurity education platform providing hands-on lab environments for learning attack techniques and defenses. Labs are run inside pre-configured virtual machines and Docker containers that simulate realistic network and system scenarios.

More information at [seedsecuritylabs.org](https://seedsecuritylabs.org/).
